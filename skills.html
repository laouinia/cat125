My Skills / Tools


Every craftsman uses a set of tools to produce his/her masterpieces. A Software Engineer uses programming languages as tools for his/her crafts. A well-versed software engineer should be able to juggle between different programming languages. Some of the most important programming languages for back-end developers are Rust and Python. Rust is a new programming language developed on the base of C/C++ without memory management, which makes it secure and fast while Python is the goto language for Artificial Intelligence. HTML, CSS, and JavaScript are the most important languages for a front-end developer. In addition to these languages, any software engineer should know at least one database programming language such as SQL.
